# PCSK9-GNN-Scaffold-Splitting

This repository contains the data, source code, model files, and outputs associated with the scaffold-based splitting workflow used for Graph Neural Network (GNN) modeling of PCSK9 inhibitors.

## Repository Structure

```text
data/      # curated dataset and scaffold splits
models/    # trained model files
outputs/   # prediction and evaluation results
src/       # source code
```

## Description

Scaffold-based splitting was employed to partition compounds into training, validation, and test sets, providing a more rigorous assessment of model generalization.

## Contents

* Curated dataset
* Scaffold-based train/validation/test splits
* GNN source code
* Trained model files
* Output results

## Reproducibility

The files provided in this repository allow reproduction of the scaffold-splitting workflow and associated machine learning experiments described in the manuscript.
